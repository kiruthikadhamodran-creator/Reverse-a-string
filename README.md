This algorithm reverses the order of words in a string while removing leading, trailing, and redundant internal spaces using O(1) extra space and O(n) time complexity.
Step 1: Define a Helper Function (reverse)Create a utility function to reverse a segment of the string in-place between two pointers, left and right.While left < right:Swap s[left] and s[right].Increment left, decrement right.
Step 2: Clean Spaces and Reverse Individual WordsUse two pointers, read_ptr and write_ptr, to process the string from left to right:Skip Spaces: Move read_ptr forward until it hits a non-space character (the start of a word).Handle Word Separation: If write_ptr is not at the start of the string (meaning at least one word has already been written), add a single space at s[write_ptr++].Copy and Mark Word: Store the current write_ptr as the start of the word.Transfer Word: Copy characters from s[read_ptr] to s[write_ptr] until a space or the end of the string is reached.Local Reversal: Immediately call reverse(s, start, write_ptr - 1) to flip the word internally (e.g., "abc" becomes "cba").
Step 3: Finalize the String LengthOnce all words are processed:Place a null terminator (\0) at the current write_ptr position to truncate any leftover characters from the original string.Step 4: Global ReversalPerform a final reversal of the entire cleaned string (from index 0 to write_ptr - 1).This shifts the words into their final reversed order and restores the correct character sequence within each word (e.g., "cba efd" becomes "dfe abc").
